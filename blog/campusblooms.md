# How I Built CampusBlooms

CampusBlooms is a full-stack flower e-commerce platform built for college students.

## Tech Stack

- React (Frontend)
- Node.js + Express (Backend)
- Razorpay (Payments)
- Twilio (WhatsApp Order Notifications)
- Deployed on Vercel + Render

## Challenges

One major challenge was integrating Razorpay and ensuring successful payment verification.

## What I Learned

- Production deployment flow
- Payment gateway integration
- Backend API structuring
